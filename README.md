# 焚诀｜A 股短线研究与持仓辅助

焚诀不是“报代码、猜涨停”的荐股机器人。它更像一个陪你复盘和做计划的交易助手：把持仓逻辑、市场情绪、个股股性、公告风险和盘中时机放在一起看，帮你想清楚现在该拿、该等、该做 T，还是该先防守。

## 它能帮你看什么

- **已经买入：** 根据你的买入价格、大致时间和持仓逻辑，判断更适合继续拿、做 T、减仓防守还是暂时观察。
- **准备做 T：** 结合底仓、可卖数量、个股容易冲高回落还是容易走 V，给出观察条件和失效条件，尽量避免反 T。
- **想买新票：** 检查竞价、板块情绪、个股地位、消息逻辑和成交可能性；条件不够时会直接建议不出手。
- **拿不准买卖时机：** 分开看 09:25、09:40、10:30 和 14:30，不把某个高开区间机械地当成买点或卖点。
- **担心突然爆雷：** 停复牌、监管问询、纪律处分、重大公告等风险，优先级高于技术形态。

## 最简单的用法

把下面这句话发给支持安装 Skill 的 Agent：

```text
请安装并使用焚诀：https://github.com/Tebio/a-stock-signal-lab/tree/main/a-stock-signal-lab
```

安装完成后，直接像平时聊天一样告诉它你的情况。最好包括：

- 股票代码或名称
- 买入价格和大致买入时间
- 持仓数量或大概几成仓
- 你当时为什么买，准备做短线还是继续拿
- 你现在最担心什么，想让它帮你判断什么

信息不全也可以问，但焚诀会把不知道的地方说清楚，不会假装拿到了你的成交记录或实时行情。

## 可以直接复制的问法

### 已经买入，判断继续拿还是防守

```text
我今天 10:30 左右以 12.60 元买了这只票，现在大约三成仓。
我原本看好的是六氟化钨涨价逻辑。请帮我检查这个逻辑硬不硬，
再结合板块情绪、个股股性和公告风险，判断明天更适合继续拿、做 T，还是先防守。
请写清楚失效条件和下一检查点。
```

### 有底仓，准备做 T

```text
我有这只票的底仓，成本 18.20 元，今天想做 T，但不想把底仓卖飞。
请先判断它最近更容易冲高回落还是走 V，再告诉我什么条件下可以考虑高抛，
什么情况下宁可不做。没有实时数据就明确告诉我需要补什么数据。
```

### 想观察一个新买机会

```text
我正在观察这只票，计划在 09:40、10:30 或 14:30 决定要不要买。
请先查公告和监管风险，再看竞价、板块强弱、底层逻辑、个股股性和成交可能性。
条件不够就直接拒绝，不要为了给答案硬找买点。
```

## 焚诀会怎么回答

一份完整结果通常会告诉你：

- 当前更像核心持有、做 T、新买观察、风险处理还是继续等待
- 建议动作以及为什么，不只给一个模糊总分
- 哪些消息和数据已经确认，哪些还没有确认
- 最多可以承担多大风险，而不是只判断方向
- 什么情况代表原来的逻辑失效
- 下一检查点是在竞价、09:40、10:30、14:30，还是下一个交易日

## 它不会做什么

- 不连接券商，不自动下单，也拿不到你的真实成交价；买价和时间需要你自己提供。
- 不承诺稳定胜率，不保证次日一定盈利，也不会把历史样本包装成未来结果。
- 没有可靠实时行情、公告来源或成交证据时，不会编造精确买卖价。
- 竞价高开 1%—3%、4%—7%、8%—10% 只是一条线索，还要结合板块、情绪、承接和这只票自己的股性。

## 自己安装到 Codex

```bash
python ~/.codex/skills/.system/skill-installer/scripts/install-skill-from-github.py \
  --url https://github.com/Tebio/a-stock-signal-lab/tree/main/a-stock-signal-lab
```

安装后重新打开 Agent 或新建会话，让它重新读取最新版说明。

更完整的使用方法、命令和策略说明见 [焚诀完整中文指南](a-stock-signal-lab/README.md)。

## 重要提醒

本项目只用于数据研究、交易复盘和决策辅助，不构成投资建议或买卖指令。最终是否交易、挂什么价格以及使用多少仓位，都应由你自己决定。
