# 焚诀 Skill

A 股短线技术研究、候选池构建、六策略筛选、个股分析和信号验证工具。

## 发给 Agent

```text
安装并使用焚诀 Skill：https://github.com/Tebio/fenjue/tree/main/fenjue
```

Codex 安装命令：

```bash
python ~/.codex/skills/.system/skill-installer/scripts/install-skill-from-github.py \
  --url https://github.com/Tebio/fenjue/tree/main/fenjue
```

安装后重启 Agent 或 Codex。

详细说明见 [fenjue/SKILL.md](fenjue/SKILL.md) 和 [fenjue/README.md](fenjue/README.md)。

## 声明

本项目仅用于数据研究和策略验证，不构成投资建议或买卖指令。
